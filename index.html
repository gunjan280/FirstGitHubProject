Welcome to the Project to be pushed in Github

Pushing a Repositry

>> git init
>> git add .
>> git commit -m “commit_message”
>> git branch -M master
>> git remote add origin “link_to_your_repo”
>> git push -u origin master

Making Changes in existing Repository

>> git add .
>> git commit -m “commit_message”
>> git push

Creating Branches

>> git branch
>> git pull origin branch_name
>> git checkout branch_name

Collaborative Work in GitHub

>> git fetch origin branch_name
>> git pull
>> git merge
